KanjiMaster

Python Quiz

Purpose and Goal:

Created to help me remember Japanese kanji meanings.

Goals and Objectives:

Facilitates kanji learning through two quiz types:

  • - Recognizing translations and selecting kanji

  • - Recognizing kanji and selecting translations

Features:

Two quiz options:

  • - Customizable quizzes with categories like food or politeness.

  • - Planned feature to add user-generated kanjis to the database.

Technologies Used:

Developed in Python 3.11 using the 'customtkinter' library. Faced challenges in linking a database with Python, resolved with online documentation and teacher assistance.

Learning Experience:

Gained insights into Python UI design and database management.

User Interaction and Feedback:

Users engage through the program window. Received feedback that influenced improvements in the user interface.

Future Development:

Upcoming updates may include sound and a feature allowing users to add their own kanjis to the database. Envisioned evolution towards a comprehensive tool with user-generated content and interactive elements

Démineur

Python Game

Purpose and Goal

Created to develop the first game in Python and enhance skills with graphical interfaces.

Goals and Objectives

Aimed to create a visually appealing and functional Minesweeper game.

Features

Standard Minesweeper game with the added option to customize the grid size and number of mines.

Personalization features allow users to create a game with a grid of their preferred size and number of mines.

Technologies Used

Developed using Python with customtkinter for the graphical user interface (GUI).

Challenges Faced

Encountered challenges in learning how to use the grid system of tkinter to create a custom-sized grid.

Overcoming Challenges

Overcame challenges by leveraging online resources for assistance.

Learning Experience

Learned how to use the grid system of tkinter and gained an understanding of what a Frame is.

User Interaction and Feedback

Users interact with MineSweeper via the GUI with their mouse. Received feedback that influenced the colors used in the GUI.

Future Development

Plans for future updates include enhancing the overall user experience (UX) and adding the ability to mark a cell as "?"

PhotoSorter

Python Tool

Purpose and Goal:

Created to automatically sort hundreds of photographs by their taken date.

Goals and Objectives:

Developed as a simple program for casual PC users to independently sort their photos.

Features:

  • - Sorting photographs (with future plans to include videos).

  • - Personalization options for sorting, allowing users to choose folder names.

Technologies Used:

Developed using Python with customtkinter for the GUI and PIL for extracting metadata from photographs.

Challenges Faced:

Encountered difficulty in retrieving accurate metadata.

Overcoming Challenges:

Sought assistance from teachers and close contacts for overcoming metadata challenges.

Learning Experience:

Learned to manipulate photographs, files, and folders using Python.

User Interaction and Feedback:

Users interact with Photosorter through the graphical user interface (GUI). Received feedback that contributed to GUI improvements and quality-of-life enhancements, such as a button to open the explorer for folder selection.

Future Development:

Plans for future updates include adding support for videos and enhancing user-friendliness.
